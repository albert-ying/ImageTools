# ImageTools
Tool to 

- (1) convert png to dark mode without losing color hue;
- 
- (2) convert pdf to png and remove the extra empty space

## Usage

```
pdf2png pdf1 pdf2 ...
```

```
png2dark png1 png2 ...
```

## Install

Download the script and change the permission

```
chmod +x pdf2png
chmod +x png2dark
```

Then put it in your path like `/usr/local/bin`
